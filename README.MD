# Update the conan settings
- `conan remote add bincrafters https://bincrafters.jfrog.io/artifactory/api/conan/public-conan`
- `conan config set general.revisions_enabled=1`

# Linux build steps:

- `bash install.sh`

# Windows build steps:

- `.\install.ps1`
